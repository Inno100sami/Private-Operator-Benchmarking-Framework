# Private Operator Benchmarking Dashboard

An interactive, browser-based performance monitoring dashboard for private water utility operators in Rwanda.

---

## Overview

This dashboard benchmarks **19 private water operators** across **217 monthly data points** spanning FY2024–2026. It transforms raw operational data from the Excel model into an interactive HTML interface with filterable charts, KPI cards, and a full data table.

## Features

- **6 dashboard views**: Commercial, Finance, Operational, Service Level, Trends, and Data Table  
- **Filter by Year, Month, and Operator** with instant chart updates  
- **Colour-coded performance indicators** (green / amber / red) aligned to sector benchmarks  
- **Trend analysis** — track any single operator's performance over time  
- **Operator summary table** with an overall status badge (Good / Moderate / Needs Attention)  
- No server required — fully static, single-file HTML

## Metrics Covered

| Category | Key Indicators |
|---|---|
| **Commercial** | Revenue, Billing Efficiency, Metering Level, Service Complaints, Arrears, Disconnection Rate |
| **Finance** | Operating Ratio, Production Costs, Energy Costs, Labour Costs % |
| **Operational** | Non-Revenue Water (NRW %), Pump Functionality, Continuity of Supply, Leakages/km |
| **Service Level** | Water Coverage, Avg Hours of Supply, Average Tariff, Complaint Response Time |
| **HR** | Staff Productivity Index, Trained Staff % |

## Publishing to GitHub Pages

1. Push this repository to GitHub  
2. Go to **Settings → Pages**  
3. Set source to **main branch / root**  
4. Your dashboard will be live at `https://<your-username>.github.io/<repo-name>/`

## Data Source

Based on **Benchmarking Dashboard V6** (April 2026) — an Excel model tracking monthly performance submissions from private operators under Rwanda's water sector regulatory framework.

## Performance Benchmarks Used

| Metric | Target |
|---|---|
| Billing Efficiency | ≥ 95% |
| Non-Revenue Water | < 20% |
| Operating Ratio | < 1.0 |
| Pump Functionality | ≥ 90% |
| Continuity of Supply | ≥ 90% |
| Water Coverage | ≥ 80% |
